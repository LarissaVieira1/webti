O que é JavaScript?

a: O JavaScript foi criado em 1995 para tornar as páginas da web interativas, permitindo que os navegadores respondessem às ações do usuário sem precisar recarregar a página;
b: No navegador, o JavaScript é executado no computador do usuário e pode acessar elementos da página, como textos e botões. Já no servidor, com ambientes como o Node.js, o código é executado antes da resposta ser enviada ao navegador, sendo usado para processar dados e acessar bancos de dados;
c: No navegador, o JavaScript funciona em uma sandbox, um ambiente isolado que limita o acesso ao sistema operacional para proteger o usuário;
d: O document é o objeto que representa a página HTML. Com ele, o JavaScript pode acessar e modificar elementos da página, como textos, imagens e botões. O console é usado para exibir mensagens e informações no console do navegador, ajudando o desenvolvedor a testar e encontrar erros no código. Já o alert exibe uma caixa de mensagem na tela para informar algo ao usuário e só continua a execução do programa depois que o botão OK é pressionado.
