Como podemos incluir CSS em documentos HTML?

Em HTML, existem três formas principais de incluir CSS para estilizar uma página:
CSS Inline: O CSS é escrito diretamente no atributo style do elemento;
CSS Interno: O código CSS é colocado dentro da tag , localizada na seção do documento HTML;
CSS Externo: O CSS fica em um arquivo separado, geralmente com a extensão .css, e é conectado ao HTML por meio da tag .
