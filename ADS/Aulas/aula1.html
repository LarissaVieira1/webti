Display dos elementos HTML

Os elementos html possuem um exibição ou display inline ou block

block - é uma exibição que incia na margem esquerda até a direita, em que podemos determinar altura e largua do elemento. Por 33padrão, no HTML, elementos com exibição block não dividem espaço na horizontal com outros elementos

inline - é uma exibição que permite que o elemento seja apresentado dentro de uma linha com outros elementos como um elemento textual. WC3 1 ola
