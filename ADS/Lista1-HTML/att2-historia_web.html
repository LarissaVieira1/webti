Linha do Tempo

Surgimento das redes de computadores

Nas décadas de 1950 e 1960, pesquisadores começaram a desenvolver formas de conectar computadores para que eles pudessem trocar informações e compartilhar recursos de maneira mais eficiente.

ARPANET

Em 1969, foi criada a ARPANET, considerada a precursora da Internet. Ela conectava universidades e centros de pesquisa, permitindo a comunicação entre computadores localizados em diferentes lugares.

TCP/IP

Em 1983, o conjunto de protocolos TCP/IP tornou-se o padrão de comunicação entre as redes. Essa mudança possibilitou que diferentes sistemas se conectassem, formando a base da Internet atual.

DNS

O Sistema de Nomes de Domínio (DNS) foi criado para facilitar a navegação na Internet. Em vez de memorizar números de IP, os usuários passaram a acessar os sites utilizando nomes como: www.exemplo.com.

World Wide Web

Em 1989, Tim Berners-Lee desenvolveu a World Wide Web (WWW), tornando possível organizar documentos por meio de hiperlinks e facilitando o acesso às informações disponíveis na Internet.

HTML

O HTML (HyperText Markup Language) foi criado para estruturar o conteúdo das páginas da Web, permitindo inserir títulos, textos, imagens, listas, tabelas e links.

HTTP

O protocolo HTTP (HyperText Transfer Protocol) é responsável pela comunicação entre o navegador do usuário e o servidor onde as páginas estão armazenadas.

Navegadores

Os navegadores são programas utilizados para acessar páginas da Web. Eles interpretam o código HTML e exibem seu conteúdo de forma organizada para o usuário.

Servidor Web

O servidor Web é o computador responsável por armazenar os arquivos de um site e enviá-los aos navegadores sempre que um usuário solicita o acesso à página.

Web 1.0 x Web Moderna

Web 1.0:
Conteúdo praticamente estático;
Pouca interação dos usuários;
Atualizações pouco frequentes.

Aplicações Web Modernas:
Conteúdo dinâmico;
Redes sociais;
Comércio eletrônico;
Sistemas online.

Principais Conceitos

Internet:
Rede mundial de computadores.
Web:
Serviço que utiliza a Internet para disponibilizar páginas.
HTML:
Linguagem de marcação utilizada na construção das páginas.
HTTP:
Protocolo responsável pela comunicação entre navegador e servidor.
