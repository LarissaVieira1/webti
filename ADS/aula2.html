CSS

O CSS é uma linguagem de estilização e definição de comportamento sobre o HTML!

Pode ser incoporado em 3 locais diferentes.

CSS externo:
CSS externo é definido utilizando a tag link no ínicio do documento HTML na seção de HEAD. < link rel = "stylesheet" href = "style.css" >
CSS incoporado a tag style:
O CSS incoporado é difinido comumente na seção HEAD,podendo ser definido também em outros locais
CSS inline:
É definido por meio da propriedade style no próprio elemento HTML, no qual ele será utiliazdo

Model Box (Modelo em Caixa)

O HTML e CSS possuem um conceito central que é o modelo em caixa, um elemento é definido dentro do outro tendo uma estrutura comum a todos elementos HTML
