Explique detalhadamente o que ocorre durante o acesso a uma página na web, desde o momento em que o usuário digita a URL no navegador até a completa exibição do site. Descreva todas as etapas envolvidas nesse processo, incluindo a resolução de nomes de domínio, a troca de mensagens entre cliente e servidor, e os principais protocolos utilizados. Certifique-se de mencionar tecnologias como DNS, HTTP/HTTPS, TCP/IP, entre outras relevantes.

Quanto acessamos uma página web no navegador, o navegador utiliza-se do protocolo DNS para transformar o domínio da URL em um endereço IP, estabelecendo uma conexão TCP/IP com o servidor. Após esse processo, o navegador envia uma requisicao HTTP/HTTPS para o servidor solicitando a página web.Por fim, o servidor responde com um código de status, e se tudo der certo(200) com o conteúdo.
